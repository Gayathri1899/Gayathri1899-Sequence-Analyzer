# Sequence Pattern Analyzer (Java)

## Project Idea
This project is a Java Swing application that generates a mathematical sequence and analyzes the pattern.  
It separates even and odd numbers and asks the user to guess the next number in the sequence.

## Formula Used

Sequence formula:

a(n) = 2^n + (n+1)^2

Example sequence:

2, 6, 13, 24, 41, 68, ...

## Features

- Generate mathematical sequence
- Separate even and odd numbers
- Interactive GUI using Java Swing
- User guesses the next number
- Program checks answer and displays result

## Technologies Used

- Java
- Swing GUI
- Eclipse IDE
